# Outils Recommandés pour la Sécurité des Mots de Passe  

## 🛡️ Gestionnaires de Mots de Passe  
- **[Bitwarden](https://bitwarden.com/)** : Gestionnaire open source et gratuit.  
- **[Dashlane](https://www.dashlane.com/)** : Pour particuliers et entreprises.  
- **[LastPass](https://www.lastpass.com/)** : Solution simple et intuitive.  

## 🔑 Générateurs de Mots de Passe  
- **[Bitwarden Password Generator](https://bitwarden.com/password-generator/)**.  
- **[LastPass Password Generator](https://www.lastpass.com/password-generator/)**.  

## 🔍 Outils d’Audit et de Sensibilisation  
- **[Have I Been Pwned](https://haveibeenpwned.com/)** : Pour vérifier si vos mots de passe ont été compromis.  
- **[Password Checker by Google](https://passwords.google.com/)**.
